# CubeOcean Help Center

Welcome to the CubeOcean Help Center. Here you can find guides, frequently asked questions (FAQ), and basic documentation for using the CubeOcean platform.

---

## Getting Started

### What is CubeOcean?
CubeOcean is a platform for managing and scaling digital services with a flexible and modular cloud-based architecture.

### How do I create an account?
Click **Sign Up**, fill in the required information, and verify your email address to activate your account.

### Is CubeOcean free?
CubeOcean offers both free and paid plans that can be adjusted based on feature requirements and usage needs.

---

## Billing

### How does billing work?
Billing is calculated based on resource usage such as compute, storage, and API requests.

### Can I cancel anytime?
Yes, you can cancel your subscription at any time through the billing dashboard.

If no renewal is made for the next period, the subscription will automatically end at the conclusion of the current active billing cycle.

---

## Account and Security

### How do I reset my password?
Click **Forgot Password** on the login page, then follow the instructions sent to your email.

### Does CubeOcean support 2FA?
Yes, CubeOcean supports two-factor authentication (2FA) to enhance account security.

---

## API

### How do I get an API key?
Go to the dashboard, open the developer section, and generate a new API key.

### Is there API rate limiting?
Yes, API rate limits depend on your subscription plan.

---

## Usage Tips

- Use the search feature to quickly find articles
- Select categories to filter topics
- Never expose your API key on the client-side (frontend)

## Support

If you cannot find the answer you are looking for, please contact the CubeOcean support team via the dashboard or official support email.
