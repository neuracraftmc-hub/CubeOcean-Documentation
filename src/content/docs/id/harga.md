## CubeOcean Cloud Harga

### Custom Build System
CubeOcean Cloud menggunakan sistem Custom Build, yaitu sistem cloud dimana server tidak dibagi dalam paket tetap, tetapi dibangun sesuai kebutuhan pengguna.

Dengan sistem ini, pengguna bisa menentukan sendiri resource server sesuai kebutuhan aplikasi.

Saat ini Custom Build masih dalam tahap pengembangan, sehingga hanya mendukung pengaturan CPU, RAM, dan Disk.

Database, Backup, dan Region belum bisa diatur secara manual. Semua itu akan diatur otomatis oleh sistem berdasarkan jumlah dan skala Custom Build yang digunakan.

Region server juga dipilih secara otomatis berdasarkan lokasi terdekat pengguna untuk memberikan performa dan latensi yang lebih optimal.

---

### CPU, RAM, and Disk Customization
Pengguna dapat mengatur resource utama server sesuai kebutuhan:

- CPU untuk performa proses aplikasi
- RAM untuk stabilitas dan multitasking
- Disk untuk penyimpanan data

Semua perubahan akan diproses otomatis oleh sistem tanpa perlu konfigurasi tambahan.

---

### Automatic Region Selection
Sistem akan memilih region server secara otomatis berdasarkan lokasi pengguna.

Tujuannya:
- Mengurangi latency
- Mempercepat akses server
- Meningkatkan stabilitas koneksi

Pengaturan region manual belum tersedia karena masih dalam tahap pengembangan.
