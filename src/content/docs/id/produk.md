# CubeOcean Cloud Produk

## Gambaran Umum

CubeOcean Cloud adalah platform cloud hosting modern yang berfokus pada performa tinggi, stabilitas sistem, serta fleksibilitas untuk berbagai kebutuhan digital. Platform ini dibangun menggunakan pendekatan infrastruktur berbasis container yang didukung oleh **Pterodactyl Panel**, sehingga memberikan kontrol penuh, efisiensi penggunaan resource, dan skalabilitas yang optimal untuk pengguna individu maupun bisnis.

CubeOcean Cloud dirancang untuk mendukung berbagai layanan hosting seperti game server, bot automation, website hosting, database hosting, serta sistem monitoring uptime yang terintegrasi dalam satu ekosistem cloud terpadu.

---

## Layanan Utama

### 1. Game Hosting Berbasis Pterodactyl

CubeOcean Cloud menyediakan layanan game hosting yang berjalan di atas panel Pterodactyl, memungkinkan pengguna mengelola server game secara real-time dengan sistem container yang ringan dan aman.

Fitur:
- Instalasi otomatis berbagai game populer
- Panel manajemen server berbasis web
- Isolasi resource per pengguna
- Kontrol start, stop, restart secara instan
- Akses file manager untuk konfigurasi server

Layanan ini cocok untuk game seperti Minecraft, Rust, ARK, CS:GO, dan berbagai game multiplayer lainnya.

---

### 2. Bot Hosting (Otomasi & Bot Discord)

CubeOcean Cloud menyediakan hosting untuk bot berbasis Node.js, Python, dan bahasa pemrograman lainnya.

Fitur:
- Deployment bot otomatis
- Auto restart saat crash
- Monitoring log secara real-time
- Proses background stabil
- Integrasi langsung dengan sistem Pterodactyl

Cocok untuk Discord bot, Telegram bot, WhatsApp bot, dan layanan otomasi lainnya.

---

### 3. Website Hosting (Panel Mirip cPanel)

Platform ini juga menyediakan layanan website hosting dengan tampilan manajemen seperti cPanel untuk memudahkan pengelolaan website.

Fitur:
- File manager berbasis web
- Support PHP, HTML, CSS, JavaScript
- Database MySQL / MariaDB
- Installer CMS satu klik (seperti WordPress)
- Dukungan SSL
- Binding domain

Layanan ini ditujukan untuk website pribadi, company profile, dan aplikasi web ringan.

---

### 4. Database Hosting

CubeOcean Cloud menyediakan layanan database hosting yang stabil dan terisolasi untuk kebutuhan aplikasi.

Fitur:
- Support MySQL / MariaDB
- Akses remote database
- Performa query tinggi
- Backup otomatis
- Koneksi database terenkripsi

Cocok untuk aplikasi web, backend mobile, dan sistem API.

---

### 5. Sistem Monitoring Uptime

CubeOcean Cloud menyediakan sistem monitoring uptime yang terintegrasi dengan infrastruktur berbasis Pterodactyl untuk memastikan layanan tetap stabil.

Fungsi:
- Monitoring status server secara real-time
- Deteksi downtime otomatis
- Notifikasi error atau crash
- Logging performa server
- Statistik uptime harian, mingguan, dan bulanan

---

## Infrastruktur Sistem

CubeOcean Cloud dibangun dengan arsitektur cloud modern yang berfokus pada:

- Virtualisasi berbasis container (Pterodactyl Wings)
- Cluster server high availability
- Isolasi resource per pengguna
- Load balancing untuk stabilitas
- Penyimpanan SSD / NVMe berkecepatan tinggi
- Optimasi jaringan latensi rendah

Semua sistem dirancang untuk meminimalkan downtime dan memaksimalkan efisiensi resource.

## Model Hosting

### Shared Hosting (Default)

Semua layanan berjalan pada sistem shared environment dengan isolasi container per pengguna untuk menjaga stabilitas, efisiensi, dan biaya yang lebih terjangkau.

### Dedicated Server (Custom Request)

Dedicated server belum tersedia sebagai paket publik standar. Namun, jika pengguna membutuhkan:
- Performa khusus
- Resource eksklusif
- Konfigurasi enterprise

Silakan menghubungi customer service untuk pembahasan dan pengaturan lebih lanjut.
