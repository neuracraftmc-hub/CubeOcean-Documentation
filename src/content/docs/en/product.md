# CubeOcean Cloud Product

## Overview

CubeOcean Cloud is a modern cloud hosting platform focused on high performance, system stability, and flexibility for a wide range of digital needs. The platform is built using a container-based infrastructure approach powered by the **Pterodactyl Panel**, providing full control, efficient resource usage, and scalable performance for both individual users and businesses.

CubeOcean Cloud is designed to support various hosting services such as game servers, bot automation, website hosting, database hosting, and integrated uptime monitoring systems within a unified cloud ecosystem.

---

## Core Services

### 1. Game Hosting Powered by Pterodactyl

CubeOcean Cloud provides game hosting services running on the Pterodactyl Panel, allowing users to manage game servers in real-time using a lightweight and secure container-based system.

Features:
- Automatic installation of popular games
- Web-based server management panel
- Isolated resource allocation per user
- Instant start, stop, and restart controls
- File manager access for configuration

This service is suitable for games such as Minecraft, Rust, ARK, CS:GO servers, and other multiplayer game servers.

---

### 2. Bot Hosting (Automation & Discord Bots)

CubeOcean Cloud offers hosting solutions for bots built with Node.js, Python, and other programming languages.

Features:
- Automatic bot deployment
- Auto-restart on crash
- Real-time log monitoring
- Stable background process execution
- Direct integration with Pterodactyl system

Suitable for Discord bots, Telegram bots, WhatsApp bots, and automation services.

---

### 3. Website Hosting (cPanel-Style Management)

The platform also provides website hosting with a cPanel-like interface for easy website management.

Features:
- Web-based file manager
- PHP, HTML, CSS, JavaScript support
- MySQL / MariaDB databases
- One-click CMS installer (e.g., WordPress)
- SSL support
- Domain binding

This service is intended for personal websites, company profiles, and lightweight web applications.

---

### 4. Database Hosting

CubeOcean Cloud provides stable and isolated database hosting services for application needs.

Features:
- MySQL / MariaDB support
- Remote database access
- High-performance query handling
- Automatic backups
- Encrypted database connections

Suitable for web applications, mobile backends, and API systems.

---

### 5. Uptime Monitoring System

CubeOcean Cloud includes an integrated uptime monitoring system built on top of its Pterodactyl-based infrastructure to ensure service stability.

Key functions:
- Real-time server status monitoring
- Automatic downtime detection
- Crash and error notifications
- Performance logging
- Daily, weekly, and monthly uptime statistics

---

## Infrastructure

CubeOcean Cloud is built on a modern cloud architecture focused on:

- Container-based virtualization (Pterodactyl Wings)
- High availability server clusters
- Per-user resource isolation
- Load balancing for stable performance
- SSD / NVMe storage for high speed
- Low-latency network optimization

All systems are designed to minimize downtime and maximize resource efficiency.

## Hosting Model

### Shared Hosting (Default)

All services run in a shared environment with isolated containers per user to ensure stability, efficiency, and affordability.

### Dedicated Server (Custom Request Only)

Dedicated resources are not available as a standard public plan at the moment. However, users requiring:
- Dedicated performance
- Exclusive server resources
- Enterprise-level configuration

can contact customer support for custom arrangements and further discussion.
