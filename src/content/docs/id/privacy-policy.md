# Privacy Policy – CubeOcean Cloud

## 1. Pendahuluan

CubeOcean Cloud (“kami”, “layanan”, atau “platform”) berkomitmen untuk melindungi privasi pengguna. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi data Anda saat menggunakan layanan CubeOcean Cloud.

Dengan menggunakan layanan kami, Anda menyetujui kebijakan privasi ini.

---

## 2. Informasi yang Kami Kumpulkan

Kami dapat mengumpulkan beberapa jenis informasi berikut:

### a. Informasi Akun

* Nama pengguna
* Alamat email
* Informasi login (hashed password)
* Data akun lainnya yang diberikan saat registrasi

### b. Data Teknis

* Alamat IP
* Jenis perangkat dan browser
* Sistem operasi
* Log aktivitas penggunaan

### c. Data Penggunaan

* Aktivitas di dashboard
* Penggunaan fitur cloud
* Akses API (jika tersedia)

### d. Data Pembayaran (jika ada)

* Status transaksi
* Metode pembayaran (diproses oleh pihak ketiga, kami tidak menyimpan data kartu secara langsung)

---

## 3. Cara Kami Menggunakan Data

Data yang dikumpulkan digunakan untuk:

* Menyediakan dan mengoperasikan layanan CubeOcean Cloud
* Meningkatkan performa sistem dan pengalaman pengguna
* Keamanan dan pencegahan penyalahgunaan sistem
* Menangani dukungan pelanggan
* Mengirimkan informasi penting terkait layanan
* Analitik internal untuk pengembangan fitur
* Pengembangan, pelatihan, dan peningkatan model kecerdasan buatan (AI), termasuk chatbot dan sistem machine learning berbasis data penggunaan dan interaksi pengguna

Data aktivitas pengguna dapat digunakan untuk membantu meningkatkan kualitas respons, akurasi, dan kemampuan sistem berbasis AI.

---

## 4. Penyimpanan dan Keamanan Data

Kami menerapkan langkah-langkah keamanan teknis dan organisasi untuk melindungi data Anda, termasuk:

* Enkripsi data sensitif
* Firewall dan proteksi server
* Monitoring aktivitas mencurigakan
* Pembatasan akses internal

Namun, tidak ada sistem yang 100% aman. Kami tidak dapat menjamin keamanan absolut dari data yang dikirim melalui internet.

---

## 5. Pembagian Data ke Pihak Ketiga

Kami tidak menjual data pribadi pengguna.

Namun, data dapat dibagikan dalam kondisi berikut:

* Penyedia layanan pihak ketiga (hosting, payment gateway, analytics)
* Kewajiban hukum atau permintaan otoritas
* Perlindungan terhadap penyalahgunaan atau pelanggaran sistem

Semua pihak ketiga wajib menjaga kerahasiaan data pengguna.

---

## 6. Hak Pengguna

Pengguna memiliki hak untuk:

* Mengakses data pribadi
* Memperbarui atau memperbaiki data
* Meminta penghapusan akun
* Menarik persetujuan penggunaan data (dalam batas tertentu)

Permintaan dapat dilakukan melalui kontak resmi CubeOcean Cloud.

---

## 7. Penyimpanan Data (Retention)

Kami menyimpan data selama akun aktif atau selama diperlukan untuk tujuan operasional layanan.

Data dapat dihapus apabila:

* Akun dihapus oleh pengguna
* Tidak ada aktivitas dalam jangka waktu tertentu
* Diperlukan untuk kepatuhan hukum

---

## 8. Privasi Anak-anak

Layanan CubeOcean Cloud tidak ditujukan untuk pengguna di bawah usia minimum sesuai hukum yang berlaku. Kami tidak secara sengaja mengumpulkan data dari anak-anak.

Jika ditemukan data anak di bawah umur, kami akan segera menghapusnya.

---

## 9. Perubahan Kebijakan

CubeOcean Cloud dapat memperbarui kebijakan privasi ini kapan saja.

Perubahan akan diumumkan melalui platform atau email resmi. Penggunaan layanan secara berkelanjutan dianggap menyetujui perubahan tersebut.

---

Jika Anda memiliki pertanyaan terkait privasi, Anda dapat menghubungi kami melalui:

* Email: [support@cubeocean.web.id](mailto:support@cubeocean.web.id)
* Website: [https://cubeocean.cloud](https://cubeocean.web.id) (jika tersedia)
