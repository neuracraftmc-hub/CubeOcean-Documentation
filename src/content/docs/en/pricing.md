## CubeOcean Cloud Pricing

### Custom Build System
CubeOcean Cloud uses a Custom Build system, where servers are not offered in fixed packages but are built based on user requirements.

This system allows users to define their own server resources according to the needs of their applications.

The Custom Build feature is still under development, so currently it only supports configuration of CPU, RAM, and Disk.

Database, Backup, and Region cannot be manually configured at this stage. All of these are automatically managed by the system based on the scale and amount of Custom Build usage.

Server region is also automatically selected based on the user’s nearest location to ensure lower latency and better performance.

---

### CPU, RAM, and Disk Customization
Users can adjust core server resources based on their needs:

- CPU for processing performance
- RAM for stability and multitasking
- Disk for data storage

All changes are processed automatically by the system without additional manual configuration.

---

### Automatic Region Selection
The system automatically selects the server region based on the user’s location.

The purpose of this system is:
- To reduce latency
- To improve access speed
- To provide more stable connectivity

Manual region selection is not available as it is still under development.
