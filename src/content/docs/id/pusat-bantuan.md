 # Pusat Bantuan CubeOcean

Selamat datang di Pusat Bantuan CubeOcean. Di sini kamu dapat menemukan panduan, pertanyaan umum (FAQ), dan dokumentasi dasar untuk menggunakan platform CubeOcean.

---

## Memulai

### Apa itu CubeOcean?
CubeOcean adalah platform untuk mengelola dan mengembangkan layanan digital dengan arsitektur cloud modular yang fleksibel dan mudah digunakan.

### Bagaimana cara membuat akun?
Klik **Daftar**, isi data yang diperlukan, lalu lakukan verifikasi melalui email untuk mengaktifkan akun kamu.

### Apakah CubeOcean gratis?
CubeOcean menyediakan paket gratis dan berbayar yang dapat disesuaikan dengan kebutuhan fitur dan penggunaan.

---

## Penagihan (Billing)

### Bagaimana sistem penagihan bekerja?
Penagihan dihitung berdasarkan penggunaan sumber daya seperti komputasi, penyimpanan, dan permintaan API.

### Apakah saya bisa membatalkan kapan saja?
Ya, kamu dapat membatalkan langganan kapan saja melalui dashboard penagihan.

Jika tidak dilakukan perpanjangan pada periode berikutnya, maka langganan akan otomatis berakhir pada akhir masa aktif saat ini.

---

## Akun dan Keamanan

### Bagaimana cara mengatur ulang kata sandi?
Klik **Lupa Kata Sandi** pada halaman login, lalu ikuti instruksi yang dikirimkan ke email kamu.

### Apakah CubeOcean mendukung 2FA?
Ya, CubeOcean mendukung autentikasi dua faktor (2FA) untuk meningkatkan keamanan akun.

---

## API

### Bagaimana cara mendapatkan API key?
Masuk ke dashboard, buka menu developer, lalu buat API key baru.

### Apakah ada pembatasan API (rate limit)?
Ya, pembatasan API tergantung pada paket langganan yang kamu gunakan.
